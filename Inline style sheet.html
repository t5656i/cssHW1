inline style sheet

inline style sheet內聯樣式表單:
主要是透過在<body>裡的單一元素中加入style屬性為其添加樣式,
style屬性中可以包含各式CSS屬性,其結構範例如下: 

                
                <!DOCTYPE html>
                <html>
                <head>
                    <meta charset="UTF-8">
                    <meta name="inline style sheet" content="width=device-width, initial-scale=1.0">
                    <meta http-equiv="X-UA-Compatible" content="ie=edge">
                    <title>Document</title>
                </head>
                <body>
                    <h1 style="text-align:center;color:red;">inline style sheet </h1>
                </body>
                </html>

上面結構範例會讓其標題inline style sheet顯示為置中紅色字體,如本篇標題。
