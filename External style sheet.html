external style sheet

external style sheet外部樣式表單:
主要是透過在<head>中建立<link>連結至外部的CSS檔案為內文添加樣式,
其結構範例如下(CSS檔請參考附檔): 

               
                <!DOCTYPE html>
                <html>
                <head>
                    <meta charset="UTF-8">
                    <meta name="external style sheet" content="width=device-width, initial-scale=1.0">
                    <meta http-equiv="X-UA-Compatible" content="ie=edge">
                    <title>Document</title>
                    <link rel="stylesheet" href="mycss/css0.css">
                </head>
                <body>
                    <h1>external style sheet </h1>
                </body>
                </html>

上面結構範例會讓其標題external style sheet顯示為置中紅色字體,如本篇標題。
