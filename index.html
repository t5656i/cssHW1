internal style sheet

internal style sheet內部樣式表單:
主要是透過在<head>中建立<style>為內文添加樣式,
<style>中用以撰寫各式CSS屬性,其結構範例如下: 

                
                <!DOCTYPE html>
                <html>
                <head>
                    <meta charset="UTF-8">
                    <meta name="internal style sheet" content="width=device-width, initial-scale=1.0">
                    <meta http-equiv="X-UA-Compatible" content="ie=edge">
                    <title>Document</title>
                    <style>
                      h1 {
                           text-align: center;
                           color: red; 
                        }
                </head>
                <body>
                    <h1>internal style sheet </h1>
                </body>
                </html>

上面結構範例會讓其標題internal style sheet顯示為置中紅色字體,如本篇標題。
